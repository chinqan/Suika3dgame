## 1.1 核心概念

Neon Shape Merge 是一款以「合成升級」為核心玩法的 **3D 物理落下益智遊戲**，靈感來自西瓜遊戲（Suika Game）。玩家從容器頂端投入 **3D 立體幾何多面體**，**相同形狀碰撞後合成為更大的下一階立體形狀**，目標是累積盡可能高的分數而不讓形狀堆出遊戲區域。

遊戲畫面以 **Three.js** 渲染 3D 場景，搭配 **Rapier3D (WASM)** 物理引擎模擬真實的 3D 碰撞與堆疊，呈現出霓虹多面體在深空容器中翻滾、碰撞、合成的立體視覺體驗。

## 1.2 設計支柱

| 支柱 | 說明 |
|------|------|
| **簡單上手，難以精通** | 單一操作（點擊/觸控）即可開始遊戲，策略深度在堆疊位置規劃與合成鏈預判 |
| **即時回饋爽感** | 每次合成觸發 3D 粒子爆炸、音效震擊、分數浮字、攝影機震動，形成即時多感官回饋 |
| **漸進節奏壓力** | 形狀隨機出現、堆疊空間持續壓縮，在輕鬆與緊張之間製造節奏張力 |
| **霓虹 3D 美學身份** | 深黑背景配高飽和度霓虹色 3D 多面體與透視地板格線，打造強烈 Cyberpunk 立體視覺識別 |

## 1.3 核心遊戲循環

```
投放 3D 形狀 → 物理落下堆疊（3D 碰撞） → 碰撞合成升級
     ↑                                      ↓
確認 NEXT 形狀 ← 獲得分數 ← 觸發 3D 視覺/音效回饋
```

### 時間維度分析

| 維度 | 時間 | 玩家體驗 |
|------|------|---------|
| **即時**（Moment-to-Moment） | 0–5 秒 | 選擇落點 → 投放 → 觀察 3D 形狀翻滾、彈跳、碰撞合成 |
| **Session**（單局） | 5–30 分鐘 | 管理堆疊高度、規劃合成鏈、追求連擊 COMBO、緊張感漸增 |
| **長期**（跨局） | 多局累計 | 挑戰個人最高分（localStorage）、排行榜社群競爭 |

## 1.4 目標玩家

| 類型 | 動機 | 遊戲提供 |
|------|------|---------|
| **休閒玩家** | 打發碎片時間、享受 3D 視覺爽感 | 簡單操作、即時 3D 粒子回饋與音效 |
| **益智玩家** | 追求最佳策略、最高分 | 合成鏈規劃、COMBO 系統、難度選擇 |
| **競技玩家** | 排行榜排名、社群競爭 | 線上排行榜、多難度挑戰、遊戲時間記錄 |

## 1.5 玩家心理與情感設計

### 單局情感弧線（Pacing Chart）

```
時間      | 活動階段          | 張力等級    | 玩家情緒
----------|-----------------|-----------|------------------
0:00-0:30 | 首次投放/探索      | ▓░░░░ Low  | 學習規則、好奇（3D 翻滾感新鮮）
0:30-2:00 | 堆疊成長          | ▓▓░░░ Med  | 信心建立、滿足感
2:00-4:00 | 空間壓縮+合成鏈    | ▓▓▓░░ High | 策略思考、緊張中帶成就感
4:00-6:00 | 大形狀合成達成     | ▓▓░░░ Med  | 爽感釋放（Release）
6:00-8:00 | 空間再壓縮/接近頂端 | ▓▓▓▓░ Peak | 焦慮、風險決策
8:00+     | Game Over        | ▓▓▓▓▓ Max  | 挫敗 → 「再來一局」動機
```

### 心流（Flow）設計

| 設計要素 | 挑戰來源 | 技巧來源 | 平衡點 |
|---------|---------|---------|-------|
| 空間管理 | 隨機形狀 + 3D 堆疊壓力 | 投放位置選擇 | 隨機性提供變化，選位提供掌控感 |
| 合成鏈 | 需同時接觸兩個相同形狀 | 預判 3D 堆疊結構 | 偶然連鎖帶來意外驚喜 |
| COMBO | 時間窗口壓力 (1200ms) | 連續合成路徑規劃 | 自然產生，不需刻意追求 |

### 學習曲線

- **0–15 秒**：理解「點擊投放形狀」的核心操作
- **1–3 局**：發現相同形狀碰撞會合成，觀察 3D 多面體的翻滾與堆疊行為
- **3–5 局**：開始有意識地安排形狀位置
- **5+ 局**：理解 NEXT 預覽的策略意義，規劃合成鏈

## 1.6 遊戲特色摘要

- 🎮 **9 種 3D 立體多面體進化系統**（Tetrahedron → Large Sphere）
- 🎯 **3 種難度選擇**（Easy / Normal / Hard）
- 💥 **3D 合成粒子與攝影機震動回饋**（Ring / Shard / Dot 三層 3D 粒子）
- 🔊 **8 種可切換音效風格**（標準 / 水晶 / 水滴 / 炸彈 / 石頭 / 宇宙 / 格鬥 / 射擊）
- 🏆 **線上排行榜**（後端 API + SQLite，Top 50）
- 📱 **跨平台支援**（桌機 + 行動裝置）
- ⚡ **Three.js WebGL2 渲染**（含 UnrealBloomPass 後處理泛光）
- 🧊 **Rapier3D WASM 物理引擎**（高效能 3D 碰撞模擬）
